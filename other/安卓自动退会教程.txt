参考文献：https://github.com/yqchilde/JDMemberCloseAccount#screenshots

1.安装python（安装的时候注意！！！一定要勾选添加到path：add python to path）
https://www.python.org/downloads/release/python-395/
如果没有勾选添加到path，要么重新安装，要么https://blog.csdn.net/qq_42278138/article/details/96178202

2.在项目根目录下打开终端执行以下命令，安装所需要的包
用cmd或者powershell cd到.\JDMemberCloseAccount-main
例如
cd C:\Users\zhaox\Desktop\1\JDMemberCloseAccount

3.安装selenium
cmd或者powersgell在项目根目录下执行
pip install selenium

4.pip3 install -r requirements.txt
如果因没有代理拉不下包，请使用国内阿里云代理，执行如下命令：
pip3 install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/

5.chrome请访问chrome://version/查看浏览器的版本
只需要保证版本号前三段一致即可，比如91.0.4472.77只需要保证91.0.4472.x就行
插件下载地址http://chromedriver.storage.googleapis.com/index.html
把下载好、解压好的exe放进drivers文件夹

6.填写config.yaml
安卓仅需要填写cookie

7.安卓手机安装macrodroid
导入extra文件夹内macrodroid文件夹里面的：监听京东退会验证码.macro

8.打开jd_wstool
根据jd_wstool里面的ip（本机电脑ip）在手机上测试通断

9.修改macrodroid的http get
把7中成功测试通过的ip，填进macrodroid里面”动作“→”HTTP GET“里面的ip

10.cmd或者powershell在项目根目录下执行python main.py

python test_main.py
http://10.10.10.104:5201/publish?smsCode=1234522
